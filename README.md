<h1>🎲 DevSorteio</h1>

<h2>Projeto simples de sorteador de números desenvolvido durante os estudos de HTML, CSS e JavaScript.</h2>

<h2>O usuário informa um número mínimo e um número máximo, e o sistema sorteia um número aleatório dentro desse intervalo.</h2>

<h3>🚀 Tecnologias utilizadas</h3>

<ul>
<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>
</ul>

<h3>📌 Funcionalidades</h3>

<ul>
<li>Inserir número mínimo</li>
<li>Inserir número máximo</li>
<li>Sortear número aleatório</li>
<li>Validação para evitar intervalo inválido</li>
</ul>

<h3>💻 Como funciona</h3>

<ul>
<li>O usuário digita o número inicial</li>
<li>Digita o número final</li>
<li>Clica no botão Sortear</li>
<li>O JavaScript gera um número aleatório entre os valores informados</li>
</ul>
